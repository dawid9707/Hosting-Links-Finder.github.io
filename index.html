Hosting Links Finder

What is it?

Hosting Links Finder is a Google Chrome extension that fetches file hosting links from websites. It supports popular file hosting services such as:

  • rapidgator.net
  • vikingfiles.com
  • turbobit.net
  • dailyuploads.net
  • mixdrop.co
  • 1fichier.com

Install Instructions

  1. Download the latest release from the GitHub page.
  2. Unzip the downloaded archive.
  3. In Chrome, go to Extensions → enable Developer Mode → click Load unpacked → select the unzipped Hosting.Links.Finder folder.
